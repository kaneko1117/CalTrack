# 環境変数ファイルポリシー

## 禁止事項

**環境変数ファイルを直接編集しないこと**

以下のファイルは直接操作禁止:
- `.env`
- `.env.local`
- `.env.development`
- `.env.production`
- `.env.test`
- その他 `.env*` パターンに該当するファイル

## 対応方法

環境変数の変更が必要な場合:

1. **変更内容を出力する**
2. **ユーザーに手動で書いてもらう**

### 出力フォーマット

```
以下の内容を `.env` ファイルに追記してください:

ENV=test
```

または

```
`.env` ファイルを以下の内容に変更してください:

ENV=test
DATABASE_URL=xxx
```

## 理由

- 環境変数ファイルには機密情報が含まれる可能性がある
- 環境ごとに異なる設定が必要
- ユーザーが意図しない変更を防ぐ
